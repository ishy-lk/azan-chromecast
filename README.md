# My Hacks Collection

A collection of useful scripts and tools.

## Projects

### [azan-chromecast](./azan-chromecast/)
Automatic prayer time notifications with Azan playback on Google Chromecast devices. Features include:
- Automatic prayer time calculation based on location
- Volume control (quieter for Fajr)
- Visual display with prayer name and background image
- Test modes for debugging

See the [azan-chromecast README](./azan-chromecast/README.md) for detailed documentation.
